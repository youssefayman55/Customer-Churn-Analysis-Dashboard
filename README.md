# 📊 Customer Churn Analysis Dashboard

An interactive data analysis project that explores customer behavior and identifies the key factors that lead to customer churn.  
Built using Python and Streamlit to provide clear business insights through visualization.

## 🎯 Project Objective :

The main goal of this project is to:
1- Analyze customer data to understand churn behavior
2- Identify patterns and factors affecting customer retention
3- Provide actionable insights for business decision-making
4- Build an interactive dashboard for data exploration

---

## 📂 Dataset Description :

The dataset contains customer-level information such as:

- Gender
- Senior Citizen status
- Tenure (subscription duration)
- Monthly Charges
- Total Charges
- Contract Type
- Payment Method
- Churn Status (Target Variable)

## 📊 Dashboard Features :

The Streamlit dashboard includes:

### 📌 Key Metrics :
1- Total number of customers
2- Churn rate percentage
3- Average monthly charges

### 📌 Interactive Filters :
1- Gender filter
2- Contract type filter

### 📌 Visual Analysis :
1- Churn distribution
2- Contract type vs Churn
3- Monthly charges vs Churn
4- Tenure vs Churn
5- Payment method impact

---

## 🧠 Key Insights :

From the analysis, we discovered:

1- Customers with **short tenure** are more likely to churn
2- **Month-to-month contracts** show higher churn rates
3- Higher **monthly charges** may increase churn probability
4- Certain **payment methods** are linked to higher churn
